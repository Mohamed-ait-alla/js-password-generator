# 🔐 Random Password Generator
A simple and secure web application for generating strong random passwords. Built using pure **HTML**, **CSS**, and **JavaScript**, this app allows users to customize passwords based on their preferences.

---

## 🚀 Live Demo
[👉 View the App](https://mohamed-ait-alla.github.io/js-password-generator/)

---

## ✨ Features
- 🔢 Generate strong random passwords instantly
- ⚙️ Customize password content:
    - Uppercase letters (A-Z)
    - Lowercase letters (a–z)
    - Numbers (0–9)
    - Symbols (!@#$%^&*, etc.)
- 📏 Adjustable password length:
    - Minimum: **8 characters**
    - Maximum: **25 characters**
- ⚡ Fast and lightweight (no external libraries)
- 📋 Copy-To-Clipboard functionality

---

## 🛠️ Technologies Used
- **HTML5** – Structure of the application
- **CSS3** – Styling and layout
- **JavaScript (Vanilla)** – Logic and interactivity

## ⚙️ Getting Started
To run the project locally:

1. Clone the repository:
```bash
git clone https://github.com/Mohamed-ait-alla/js-password-generator.git
```

2. Navigate into the project folder:
```bash
cd js-password-generator
```
3. Open `index.html` in your browser.

---

## 🔒 Security Note
This app runs entirely in your browser. No passwords are stored, shared, or transmitted anywhere.
